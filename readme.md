# ：EduAPP - 在线教育学习平台

## 1. 项目简介 
主要是用来学习多端开发及打包的小项目。

基于 UniApp 开发的跨平台在线教育应用，支持课程浏览、视频播放、互动答题等功能，适配小程序、H5 和 App 端。
当前开发进度：搭建框架->首页布局绘制


## 2. 技术栈
- **前端框架**：UniApp + Vue.js  
- **UI 组件库**：uView UI  
- **后端接口**：模拟接口，采用本地数据
- **开发工具**：HBuilderX
- 目录结构如下：
├── pages/            # 页面文件
├── components/       # 自定义组件
├── static/           # 静态资源
├── App.vue           # 应用入口
├── main.js           # 主入口文件
└── README.md         # 项目说明
